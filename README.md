# Opencode sandbox

Don't want to train AI with your ssh private key?
Want to keep your GPG key safe?
Think your emails are for your eyes only?

Then this repository is for you.

This scripts uses bubblewrap to confine Opencode.

## Before use

- install opencode and connect to your models provider and MCP servers;
- install bubblewrap;
- install opencode-sandbox in your PATH.

## Usage

Move to some directory.
Call `opencode-sandbox`.
